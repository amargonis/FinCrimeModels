# Models

This repository contains Fincrime models for the Teradata AOA.

# Available Models

Supervise Model for Bustout Fraud (see [here](https://consulting-github.teradata.com/FinancialIndustrySolutions/FinCrimeModels/tree/master/model_definitions/b9f8d1c3-6043-44b8-8c5b-20dcd7cf03a9))

MLE-Based Supervised Model for AML (see [here](https://consulting-github.teradata.com/FinancialIndustrySolutions/FinCrimeModels/tree/master/model_definitions/a08b6c9e-07b6-4720-972e-438db3a398e2))


# Adding new Models

Use the aoa cli (see [here](https://pypi.org/project/aoa/))

    aoa add

## Running Models Locally

Use the aoa cli (see [here](https://pypi.org/project/aoa/))

    aoa run -h
